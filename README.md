## 🚀 Setup & Run

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Setup MySQL Database
```bash
mysql -u root -p < database.sql
```

### 3. Train the AI Model
```bash
python train_model.py
```
This generates `fraud_model.pkl`. The `/accuracy` page can also generate the model, confusion matrix, accuracy chart, and cached metrics automatically from `fraud_output.csv`.

### 4. Run the Portal
```bash
python app.py
```
Open: **http://localhost:5000**

## 🌐 Deploy Live

This is a Flask + MySQL project, so use a Python backend host. Railway is the easiest option because it can host both the Flask app and a MySQL database. Render also works if you connect an external MySQL database such as PlanetScale, Aiven, or Railway MySQL.

### GitHub Upload
```bash
git init
git add .
git commit -m "Initial NSFP portal"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/nsfp-portal.git
git push -u origin main
```

### Railway / Render Settings

Build command:
```bash
pip install -r requirements.txt
```

Start command:
```bash
gunicorn wsgi:app
```

Environment variables:
```bash
DB_HOST=your-mysql-host
DB_USER=your-mysql-user
DB_PASS=your-mysql-password
DB_NAME=gov_ai_fraud
SECRET_KEY=change-this-secret-key
```

Before opening the live portal, import `database.sql` into the live MySQL database.

## 🔐 Security Features (v2.0)
- SHA-256 password hashing
- Session-based authentication with login_required decorator
- HTTPOnly session cookies
- Environment variable support for DB credentials
- Input validation on all form fields
- Prediction save errors are reported instead of shown as successful
- SQL injection prevention via parameterized queries

## 🤖 AI Model
- **Algorithm:** IsolationForest (anomaly detection)
- **Features:** Age, Annual Income, Schemes Taken
- **Train/Test:** 80/20 split with stratification
- **Output:** fraud_predicted (1=Fraud, 0=Genuine)

## 🎨 UI Features (v2.0)
- Dark/Light mode with auto night detection
- Animated stat counters
- Collapsible sidebar
- Live clock in topbar
- Table search/filter
- Responsive design (mobile-ready)
- Bar/Donut chart switcher
